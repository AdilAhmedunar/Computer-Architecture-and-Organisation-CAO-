# 💻 Computer Architecture & Organization (CAO)

The course introduces the internal structure and operation of computer systems, including processors, memory hierarchy, instruction sets, and system organization.

## 📚 Course Topics

Key areas covered in this course include:

- Computer system structure and evolution  
- Computer performance and Amdahl’s Law  
- Computer components and bus interconnection  
- Memory hierarchy and cache memory  
- Computer arithmetic and number representation  
- Instruction sets and addressing modes  
- Processor organization and instruction cycle  
- Instruction pipelining (RISC vs CISC)  
- Control unit operation  
- Parallel and multiprocessor systems
